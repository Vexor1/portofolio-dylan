Top! Laten we jouw digitale portfolio online zetten met je CV zichtbaar via GitHub Pages – dat is gratis, betrouwbaar en perfect voor een portfolio.

✅ Stap-voor-stap: Portfolio online zetten met je CV op GitHub Pages

1. 📦 Maak een GitHub-account aan

Ga naar https://github.com en maak een gratis account aan als je die nog niet hebt.

2. 🗂️ Maak een nieuwe repository

  • Klik op "New repository"

  • Geef het bijvoorbeeld de naam: portfolio-dylan

  • Vink aan: ✅ Public

  • Vink aan: ✅ Add a README file

  • Klik op "Create repository"

3. 💻 Upload je bestanden

Klik in je nieuwe repository op "Add file" → "Upload files"
Upload daar de volgende bestanden:

  • Je index.html (het HTML-bestand dat ik hierboven voor je heb aangepast)

  • Je CV-bestand, bijvoorbeeld cv-dylan-boonstra.jpg (of PDF, dan verwijs je in je HTML naar cv-dylan-boonstra.pdf en gebruik je een <a>-link i.p.v. een <img>)

Bijvoorbeeld in je HTML:

html
<p><a href="cv-dylan-boonstra.pdf" target="_blank">Klik hier om mijn CV te bekijken (PDF)</a></p>

4. 🌐 Activeer GitHub Pages

  • Ga naar het tabblad "Settings" van je repository

  • Scroll naar beneden naar "Pages"

  • Kies bij "Source": Deploy from a branch

  • Selecteer branch: main, en folder: / (root)

  • Klik op "Save"

GitHub genereert nu een link zoals:
